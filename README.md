# SLPA — Vehicle Management UI

A web-based front-end for the **Sri Lanka Ports Authority (SLPA)** vehicle tracking system, built by Group SysCall (University of Moratuwa, Faculty of IT, 2013).

## About

This is the web UI companion to the SLPA Windows application. It allows staff to log in and view/manage records of drivers who leave the port, including job time and destination tracking.

## Features

- **Login system** — role-based authentication (Admin / User) via AJAX
- **Driver records** — view and manage driver job information
- **Destination tracking** — records destination and time for each job
- **Responsive UI** — built with custom CSS and jQuery

## Tech Stack

- **Backend:** PHP
- **Frontend:** HTML, CSS, JavaScript (jQuery)
- **Database:** MySQL

## Prerequisites

- A web server with PHP support (e.g. Apache/XAMPP)
- MySQL Server running locally
- Database: `syscall`

## Getting Started

1. Clone the repository into your web server's root directory (e.g. `htdocs/slpa`)
2. Set up the MySQL database and configure credentials in the PHP files
3. Open `index.php` in your browser to access the login page
